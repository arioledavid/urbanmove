# Urban Move Logistics — Waitlist Landing Page

A high-converting, premium waitlist landing page for **Urban Move Logistics**, a pure logistics aggregator (not a marketplace). Built with Next.js 14 (App Router), TypeScript, Tailwind CSS, and Framer Motion.

## Tech Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion**

## Brand

- **White** — primary background
- **#e0110c** — primary CTA and accents
- **#fcb900** — subtle highlights (progress bar, underlines)

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Structure

- **Navbar** — Sticky, shadow on scroll, “Join Waitlist” CTA
- **Hero** — Headline, subtext, CTAs, animated delivery card
- **Social Proof** — Count-up numbers (businesses & riders)
- **About** — Pure logistics positioning
- **For Businesses** — 6 feature cards + CTA
- **For Riders** — Bullets + CTA + yellow highlight
- **How It Works** — 3 steps with red icons and yellow underlines
- **Waitlist** — Multi-step form (Business/Rider toggle, progress bar, step transitions)
- **Vision** — Short statement
- **Footer** — Links and copyright

Form flow: Step 1 (name, email, phone) → Step 2 (business or rider fields) → Step 3 (confirmation + referral link).
